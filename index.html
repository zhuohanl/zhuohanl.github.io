Email Spam Detection by Classifiers (KNN, Bernoulli Naive Bayes and SVM kernels)

Source: HW2.1 for Unstructured Data Analytics course in Carnegie Mellon University
Instructor: George Chen

My work:
https://gist.github.com/zhuohanl/5e24371130d624d31ca277a78c6b17af

This project:

  • uses a training dataset which contains 1488 ham and 3659 spam emails
  • creates feature matrix which is a binary feature matrix which indicates if a word is present in an email (no duplicates)
  • applied KNN, Bernoulli Naive Bayes and SVM LinearSVC classifiers (with 5-fold cross validation) on the feature matrix
  • compare the results of the three classifiers

Then:

  • improves the accuracy by turning the feature matrix into one which counts the occurrence of a word in an email (with duplicates) and re-applying KNN classifier

Latent Dirichlet Allocation (LDA) Topic Modelling on Mobile Applications

Source: HW2.1 for Unstructured Data Analytics course in Carnegie Mellon University
Instructor: George Chen

My work:
https://gist.github.com/zhuohanl/c7b4676290aae3b395776cb5fa37e206

This project:

  • uses a dataset which contains ~180,000 applications from the Android app store, including each app’s name, description and permissions requested
  • applies sklearn LatentDirichletAllocation on the dataset, regarding each app as a “document”, its requested permissions as “words”, and app purposes as “topics”, including:
    • discovering 10 topics
    • listing the top 10 “words” in each topic
    • for each topic, listing the top 3 apps with the highest probability assigned to that topic
    • finding the most probable topic for the top 10 apps with the most ratings
    • applying t-SNE to reduce the 10 topics to 2-D and visualizing by plotting

Use spaCy to perform Frequency Analysis and Co-occurrence Analysis on text corpus

Source: HW1 for Unstructured Data Analytics course in Carnegie Mellon University
Instructor: George Chen

My work:
https://gist.github.com/zhuohanl/9b69ebf1caab7dee78d423187d45dfd1

This project includes three tasks:

Task 1: some simple python practice as warm-up

Task 2:

  • use spaCy package to perform Frequency Analysis on 100 most popular books downloaded from Gutenburg Project (https://www.gutenberg.org/browse/scores/top),
  • visualize by plotting, and
  • check whether it is in accordance with Zipf’s law.

Task 3:

  • use spaCy package to perform Co-occurence Analysis on Reuters corpus which contains more than ten thousand newspaper articles
  • visualize by plotting and listing top 10
  • list the top 50 GPE (countries, cities, states) entities that have the highest Pointwise Mutual Information (PMI) values with regard to the ORG (organization) entity ‘opec’
